🍺 爱尔法鲁酒吧问题

1994年，经济学家 W. Brian Arthur 提出了这个思想实验：

圣菲镇有一家酒吧，每周四有现场音乐演出。对100位居民来说，去了就很开心——除非太拥挤。于是大家都想：「如果今晚人少于60人我就去，否则不去。」

但每个人只能靠历史经验猜测今晚会有多少人，没有人互相商量。

✨ 涌现：混沌中的秩序

这里有一个悖论：如果所有人都预测「今晚人很少」→ 大家都去 → 实际很挤 → 预测错了。不存在一个「大家都同意」的预测。

但神奇的是：尽管每人各行其是，出勤人数会自发地稳定在阈值附近——宏观秩序从微观混沌中涌现，没有任何人设计或协调。
